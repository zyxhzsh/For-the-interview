**描述**

假设你正在爬楼梯。需要 n 阶你才能到达楼顶。

每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？

注意：给定 n 是一个正整数。

#### 思路一

动态规划

用f(x)表示爬到第x级台阶的方案数。

f(x)=f(x−1)+f(x−2)。

```java
class Solution {
    public int climbStairs(int n) {
        if(n<=3)
            return n;
        int p=1, q=2, r=3;//f(1)=1,f(2)=2,f(3)=3
        for(int i=4;i<=n;i++)
        {
            p=q;
            q=r;
            r=p+q;
        }
        return r;
    }
}
```
